List-group


Ni borde vara hyfsat bekväma vid att använda listor vid det här laget, men Bootstrap har lite egna klasser som man kan använda för listor.

Till exempel behöver inte en lista i Bootstrap vara ett list element i html (ul t.ex.), utan kan appliceras på andra element för att få dem att verka som listor. Ett vanligt exempel är att man applicerar listklassen direkt på länkar.
